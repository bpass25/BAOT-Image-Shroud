# BAOT-Image-Shroud


BAOT Image Shroud (V1.0)

Tool: Steganography Tool

Team: BAOT

Language: Python

💡 About the Tool: A simple yet powerful tool that uses EOF (End of File) technology to hide text and secret messages within image files (JPG, PNG, etc.). The key feature is that the image remains fully functional and viewable, making the hidden data completely invisible to the naked eye and traditional scanning systems.
